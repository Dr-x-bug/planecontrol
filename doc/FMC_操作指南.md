# FMC 飞行管理计算机 — 操作指南

## 一、页面导航

| 功能键 | 跳转页面 | 说明 |
|---|---|---|
| `INIT REF` | INIT/REF INDEX | 系统主页 / 导航数据状态 |
| `RTE` | RTE | 航路配置（起降机场 / 航班号 / 航路点） |
| `CLB` | CLB | 爬升性能参数 |
| `CRZ` | CRZ | 巡航性能参数 |
| `DES` | DES | 下降性能参数 |
| `DEP ARR` | DEP/ARR INDEX | 离场 / 进场程序选择 |
| `LEGS` | LEGS | 航段列表 |
| `HOLD` | HOLD | 等待程序 |
| `PROG` | PROG | 飞行进度 |
| `PREV PAGE` | 上翻一页 | RTE / LEGS 页面内翻页 |
| `NEXT PAGE` | 下翻一页 | RTE / LEGS 页面内翻页 |

## 二、行选键 (LSK) 操作

屏幕左右各 6 个行选键（L1~L6、R1~R6），按下将草稿栏内容填入对应行。

### 2.1 RTE 航路页面

| 行选键 | 功能 | 操作方式 |
|---|---|---|
| L1 | 设置起飞机场 (ORIGIN) | 输入机场 ICAO 码 → 按 L1 |
| R1 | 设置目的地机场 (DEST) | 输入机场 ICAO 码 → 按 R1 |
| L2 | 设置航班号 (FLT NO) | 输入航班号 → 按 L2 |
| L5/R5 | 显示上/下一页 | 直接按 L5/R5 |
| L4 | ACTIVATE / EXEC | 确认激活航路 |

> ⚠️ 输入的机场必须在 `apt.dat` 数据库中存在（AVL 树校验），否则提示 `NOT IN DB`

### 2.2 INIT/REF 主页

| 行选键 | 功能 |
|---|---|
| L1 | 进入 IDENT 状态页（导航数据库版本） |
| L5 | 从 IDENT 返回主页 |

### 2.3 DEP/ARR 页面

| 行选键 | 功能 |
|---|---|
| R1 | 切换 **离场 (DEP)** ↔ **进场 (ARR)** 模式 |
| L1 | 重置当前选择 |

**离场流程 (KSEA → DEP)：**
1. 选择跑道 → 16L/16C/16R(左) 或 34L/34C/34R(右)
2. 选择 SID 程序 → SEA5 / SUMMA2 / BANGR9 / HAROB6 / MOUNT7
3. 选择过渡点 (TRANS)

**进场流程 (KBFI → ARR)：**
1. 选择跑道 → 13L/13R(左) 或 31L/31R(右)
2. 选择 STAR → GLASR1 / OLM2 / CHINS3
3. 选择进近程序 (APPR) → ILS13L / ILS13R / RNAV31L / RNAV31R

## 三、数据输入

### 3.1 草稿栏 (Scratchpad)
- 屏幕底部 `< >` 之间为输入栏
- 使用字母键 (A-Z)、数字键 (0-9)、`.` `/` `+/-` 输入

### 3.2 编辑键

| 按键 | 功能 |
|---|---|
| `CLR` | 按一次：删除一个字符；已清空时再按：清除全部 |
| `DEL` | 进入删除模式，配合行选键删除对应行内容 |
| `SP` | 输入空格 |
| `/` | 输入斜杠（航路格式） |

### 3.3 EXEC 执行键
- 输入参数后按 `EXEC` 确认，蓝灯亮起表示待执行
- 再次按 `EXEC` 执行同步（航路数据将同步到 ND 导航显示器）
- CLB/CRZ/DES 页面按 EXEC 将草稿栏值填入当前选中行，并进行边界校验（高度 1000-41000ft、速度 100-340kt、CI 0-500）

## 四、翻页操作

| 方式 | 操作 |
|---|---|
| 物理按键 | 按 `PREV PAGE` 上翻 / `NEXT PAGE` 下翻 |
| 行选键 | RTE 页面按 L6 (上翻) / R6 (下翻) |
| 适用页面 | RTE（航路点列表）、LEGS（航段列表） |

翻页后标题栏右侧显示 `当前页/总页数`。

---

## 五、CLB / CRZ / DES 参数编辑

以 CLB 页面为例（CRZ/DES 同理）：

1. 在草稿栏输入数值（如 `290`）
2. 按对应行选键（如 L1 = SPD RESTR）
3. 按 `EXEC` 键确认
4. 值将更新到对应参数行

### 校验规则

| 参数 | 有效范围 |
|---|---|
| 高度 (FL/ALT) | 1000 ~ 41000 ft |
| 速度 (SPD) | 100 ~ 340 kt |
| 成本指数 (CI) | 0 ~ 500 |

---

## 六、快捷操作总结

| 操作 | 按键组合 |
|---|---|
| 清空草稿栏 | `CLR`（长按或连按） |
| 清除某行 | `DEL` → 按对应行选键 |
| 设置起飞机场 | 输入 `KSEA` → 按 RTE 页 L1 |
| 设置目的地 | 输入 `KBFI` → 按 RTE 页 R1 |
| 选择离场程序 | DEP ARR → 选跑道 → 选 SID → 选过渡点 |
| 查看导航数据库 | INIT REF → L1 |
| ESC 退出 | 按 `ESC` 键 |
